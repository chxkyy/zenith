# 菜单管理功能说明

## 1. 功能概述

菜单管理模块提供系统导航菜单的配置功能，支持树形结构展示、新增、编辑、删除、禁用/启用等操作。本次更新新增了在目录节点下快速新增子菜单的功能。

---

## 2. 菜单树操作

### 2.1 新增菜单按钮

位于左侧菜单树上方，用于新增菜单。

| 选中节点类型 | 按钮状态 | 行为 |
|------------|---------|------|
| DIR（目录） | ✅ 蓝色，可点击 | 打开新增菜单弹窗，父菜单字段隐藏，自动设为当前选中节点 |
| MENU（菜单） | ⚪ 灰色，置灰 | 不可点击 |
| 未选中任何节点 | ⚪ 灰色，置灰 | 不可点击 |

### 2.2 右键菜单操作

在菜单树节点上右键，弹出操作菜单。

#### 右键点击目录（DIR）节点

```
┌─────────────────────┐
│ ➕ 新增子菜单       │  ← 显示，可点击
│ ✏️ 编辑            │
│ 👁 禁用/启用       │
│ ─────────────────── │
│ 🗑 删除            │
└─────────────────────┘
```

#### 右键点击菜单（MENU）节点

```
┌─────────────────────┐
│ ✏️ 编辑            │  ← 没有"新增子菜单"按钮
│ 👁 禁用/启用       │
│ ─────────────────── │
│ 🗑 删除            │
└─────────────────────┘
```

#### 操作说明

| 操作 | 说明 |
|------|------|
| 新增子菜单 | 仅目录类型显示，点击后打开新增弹窗，父菜单自动设为当前节点 |
| 编辑 | 打开编辑弹窗，修改当前节点信息 |
| 禁用/启用 | 切换当前节点的启用状态 |
| 删除 | 删除当前节点及其子菜单、关联权限 |

---

## 3. 新增/编辑菜单弹窗

### 3.1 新增子菜单模式（从右键菜单进入）

当通过右键菜单的"新增子菜单"进入时：

- **父菜单字段**：隐藏，自动设为当前右键点击的目录节点
- **其他字段**：正常填写

### 3.2 新增菜单模式（从顶部按钮进入）

当通过顶部"新增菜单"按钮进入时：

- **父菜单字段**：显示下拉框，默认为当前选中的目录节点，可修改
- **其他字段**：正常填写

### 3.3 编辑模式

- **父菜单字段**：显示下拉框，显示当前父菜单，可修改
- **其他字段**：显示当前节点的信息，可修改

### 3.4 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 菜单名称 | 文本 | ✅ | 菜单显示的名称 |
| 菜单类型 | 下拉框 | ✅ | 目录（DIR）或菜单（MENU） |
| 父菜单 | 下拉框 | - | 选择父级目录，仅显示目录类型的菜单 |
| 路由路径 | 文本 | ✅ | 前端路由路径，如 `/dashboard` |
| 图标 | 文本 | - | 菜单图标名称 |
| 排序 | 数字 | ✅ | 菜单显示顺序，数字越小越靠前 |
| 状态 | 按钮 | ✅ | 启用（1）或禁用（0） |
| 备注 | 文本域 | - | 备注信息 |

## 4. 业务规则

### 4.1 菜单类型规则

- **目录（DIR）**：可作为父菜单，下面可以挂载子菜单
- **菜单（MENU）**：不可作为父菜单，是具体的页面菜单

### 4.2 新增子菜单规则

- 只有目录类型的节点才能新增子菜单
- 新增子菜单时，父菜单自动设为当前目录节点
- 新增子菜单时，父菜单字段不显示

### 4.3 删除规则

- 删除目录时，会同时删除其下的所有子菜单
- 删除菜单时，会同时删除其关联的功能权限和字段权限
- 删除操作不可恢复，需确认后执行

---

## 5. 操作流程示例

### 5.1 在"系统管理"目录下新增"日志管理"菜单

1. 在菜单树中选中"系统管理"（DIR类型）
2. 点击顶部"新增菜单"按钮，或右键点击"系统管理"选择"新增子菜单"
3. 在弹窗中填写：
   - 菜单名称：日志管理
   - 菜单类型：菜单
   - 路由路径：/system/logs
   - 图标：FileText
   - 排序：3
   - 状态：启用
4. 点击"保存菜单"完成新增

### 5.2 修改已有菜单

1. 在菜单树中选中要修改的菜单
2. 右键点击，选择"编辑"
3. 在弹窗中修改相关信息
4. 点击"更新菜单"完成修改

### 5.3 禁用菜单

1. 在菜单树中选中要禁用的菜单
2. 右键点击，选择"禁用"
3. 菜单状态变为禁用，前端将不显示该菜单

---

## 6. 注意事项

1. **只有目录类型才能新增子菜单**：菜单类型的节点不能作为父菜单
2. **父菜单下拉框仅显示目录类型**：选择父菜单时，只能选择目录类型的菜单
3. **删除操作需谨慎**：删除会级联删除子菜单和关联权限，操作不可恢复
4. **菜单树默认全部展开**：方便用户查看完整的菜单结构
